# NativeScript-Based-Apps

Welcome to the **NativeScript Multi-App Repository**! This repository contains multiple mobile applications built using the [NativeScript](https://nativescript.org/) framework. Each app is developed to run on both Android and iOS platforms with a shared codebase using JavaScript/TypeScript.
